# Water Distribution Data Engineering Project
## Project Overview
This project simulates a real-world water distribution business pipeline.
It focuses on end-to-end data flow — from customer registration and order
placement to delivery tracking, payment recording, and automated ETL
processing that generates daily and monthly business summaries.

## Tech Stack
- **Database:** MySQL
- **Backend:** Python, FastAPI
- **ETL:** Custom Python scripts
- **API Documentation:** Swagger UI (auto-generated by FastAPI)

## Architecture
Customer/Worker → FastAPI API → MySQL Database (OLTP)
↓
ETL Scripts run daily
↓
Summary Tables (OLAP Layer)
daily_summary / monthly_summary

## Database Schema
| Table | Description |
|-------|-------------|
| `customers` | Stores customer details |
| `orders` | Tracks all orders placed |
| `payments` | Records payment transactions |
| `deliveries` | Tracks delivery status and remarks |
| `daily_summary` | Aggregated daily business metrics (OLAP) |
| `monthly_summary` | Aggregated monthly business metrics (OLAP) |

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /customer/register | Register a new customer |
| POST | /customer/insert_order | Place a new order |
| POST | /customer/payments | Record a payment |
| POST | /worker/delivery_update | Update delivery status |
| GET | /customer/order | Get all orders with customer details |
| GET | /client/order | Get orders filtered by customer ID |

## ETL Pipeline
update_order_status.py
→ Reads deliveries table
→ Updates orders.status to DELIVERED automatically
daily_summary_etl.py
→ Aggregates previous day's orders, revenue, customers
→ Inserts into daily_summary table
monthly_summary_etl.py
→ Aggregates previous month's daily summaries
→ Inserts into monthly_summary table

## How to Run
1. Clone the repository
2. Install dependencies:
```bash
   pip install fastapi uvicorn mysql-connector-python faker
```
3. Set up MySQL database:
```bash
   mysql -u root -p < sql/schema.sql
```
4. Run the API:
```bash
   uvicorn app:app --reload
```
5. Visit `http://127.0.0.1:8000/docs` for Swagger UI
